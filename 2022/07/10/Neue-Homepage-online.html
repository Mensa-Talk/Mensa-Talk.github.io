Verwendete Tools

Markdown

Markdown ist eine Auszeichnungssprache. Das heißt, neben dem eigentlichen Inhalt wird das Format, also das Aussehen des Inhaltes auch gleich zusammen mit dem Inhalt in den Text geschrieben. Im Prinzip funktionieren html und latex Dokumente genau so. Markdown ist aber besonders einfach und eignet sich damit besonders für einfache Text die schnell “runtergeschrieben” werden und dabei halbwegs OK aussehen sollen.

  • Markdown auf Wikipedia

Jekyll static Site generator

Konvertiert Markdown Files in HTML Code. Dabei werden auf den Inhalt diverse Formatvorlagen angewandt. Die generiete Seite ist statisch. Es werden keine server- oder clientseitige Skripte Verwendet. Man benötigt also keine Tools wie Wordpress etc.

  • Jekyll site generator

Github Pages

Github ist eigentlich ein Service um Code zu versionieren und die Zusammenarbeit mit anderen zu erleichtern. Github Pages wiederrum ist ursprünglich dazu gedacht eine Homepage für die Dokumenation von Softwareprojekten zu bieten.
